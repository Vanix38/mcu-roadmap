# Logos studios

Dépose ici les logos des studios. Tant qu’un fichier est absent, la pastille texte (`MS`, `FOX`…) s’affiche.

## Nommage

Utilise l’id studio (champ `studio` dans `mcu.json`) :

| Fichier | Studio |
|---------|--------|
| `marvel.svg` (ou `.webp` / `.png`) | Marvel Studios |
| `fox.svg` | 20th Century Fox |
| `sony.svg` | Sony Pictures |
| `sony-animation.svg` | Sony Animation (Spider-Verse) |
| `netflix.svg` | Netflix |
| `abc.svg` | ABC |
| `freeform.svg` | Freeform |

Extensions essayées dans l’ordre : `.svg` → `.webp` → `.png`

## Conseils

- Format carré ou paysage compact (idéalement ~64–128 px)
- Fond transparent recommandé (SVG / PNG)
- Contraste lisible sur fond sombre
